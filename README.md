# Aplicação de Tarefas

Uma aplicação moderna de gerenciamento de tarefas desenvolvida com React, Vite e Context API.

## Sobre o Projeto

Este projeto foi desenvolvido como projeto da certificação 2 da trilha 3 do DEVstart. É uma aplicação completa para gerenciar tarefas (to-do list) com interface intuitiva e funcionalidades CRUD completas.

## Funcionalidades

- Adicionar novas tarefas
- Visualizar lista de tarefas
- Editar tarefas existentes
- Excluir tarefas
- Marcar tarefas como concluídas
- Persistência de dados com localStorage
- Notificações interativas com toasts
- Interface responsiva e moderna

## Tecnologias Utilizadas

- **React 19** - Biblioteca JavaScript para construção de interfaces
- **Vite** - Build tool moderna e rápida
- **React Router DOM** - Gerenciamento de rotas
- **Context API** - Gerenciamento de estado global
- **React Hot Toast** - Notificações elegantes
- **React Icons** - Biblioteca de ícones
- **LocalStorage** - Persistência de dados

## Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/flaviare1s/aplicacao-de-tarefas.git
```

2. Instale as dependências:

```bash
npm install
```

3. Execute o projeto em modo de desenvolvimento:

```bash
npm run dev
```

4. Acesse no navegador:

```
http://localhost:5173
```

## Scripts Disponíveis

- `npm run dev` - Inicia o servidor de desenvolvimento
- `npm run build` - Cria a build de produção
- `npm run preview` - Visualiza a build de produção
- `npm run lint` - Executa o linter

## Requisitos Implementados

- Configuração do ambiente com Vite
- React Router para navegação entre páginas
- Context API para gerenciamento de estado global
- Hooks (useState, useEffect, useContext)
- CRUD completo de tarefas
- Persistência com localStorage
- Componentização
- Interface moderna e responsiva
- Feedback visual com toasts
